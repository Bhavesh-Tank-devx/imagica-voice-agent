CONFIRM ADDRESS CITIES
For these cities the branch address must be confirmed via pin code before booking.
Ask the customer for their 6-digit pin code, then call get_closest_branches.
Cities: Jaipur, Lucknow, Kochi, Indore, Visakhapatnam, Navi Mumbai, Thane, Noida, Gurugram, Faridabad, Jalandhar.


CONSULTATION FEES
- Basic / General consultation: Free
- Detailed medical evaluation by doctor: Rs. 750
The basic consultation is always free. The Rs. 750 fee only applies if the doctor determines a detailed medical evaluation is required.


BRANCH SUGGESTION RULES
- 1 branch city: state the branch name directly.
- 2-3 branch city: name all branches.
- 4-8 branch city: ask which part of the city, then suggest 2-3 nearby branches.
- 9+ branch city (Mumbai, Delhi, Bengaluru): ask which area first, then suggest the 2 nearest branches.
- Unknown city or Confirm Address city: ask for 6-digit pin code, call get_closest_branches.
